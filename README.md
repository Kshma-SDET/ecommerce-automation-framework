A production-ready test automation framework built with Selenium Java, 
following Page Object Model design pattern with CI/CD integration.

## Tech Stack
- **Language:** Java 11
- **Automation:** Selenium WebDriver 4.18
- **Test Framework:** TestNG
- **Build Tool:** Maven
- **CI/CD:** GitHub Actions
- **Design Pattern:** Page Object Model (POM)
- **Reports:** Extent Reports
